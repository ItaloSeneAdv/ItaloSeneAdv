# Ítalo Sêne Advocacia - Site Oficial

Este é o site oficial do escritório de advocacia **Ítalo Sêne Advocacia**, especializado em **Direito Agrário, Civil e de Trânsito**.

## 🚀 Funcionalidades
- Página de apresentação
- Contato direto via **WhatsApp**
- Formulário dinâmico

## 📁 Estrutura do projeto
```
/ (Raiz do projeto)
│── index.html       # Página principal
│── styles.css       # Estilos do site
│── script.js        # Interatividade (formulário)
│── whatsapp-icon.png # Ícone do WhatsApp
│── README.md        # Documentação
│── .gitignore       # Arquivos ignorados pelo Git
│── CNAME (Opcional) # Domínio personalizado
```
